A **podridão de raiz** é uma das principais causas de morte de plantas em vasos. O pior? Os sintomas parecem com falta de água, levando muitos iniciantes a regar MAIS - e matar a planta de vez. Aprenda a identificar e agir a tempo.

## O Que é Podridão de Raiz?

Podridão de raiz é a morte e decomposição das raízes, geralmente causada por:

1. **Excesso de água** (causa #1 em vasos!)
2. **Fungos oportunistas** (Pythium, Phytophthora, Fusarium)
3. **Falta de drenagem**
4. **Solo muito compactado**

Quando as raízes apodrecem, a planta não consegue absorver água nem nutrientes - mesmo que o solo esteja úmido.

---

## Como Identificar Podridão de Raiz

### Sintomas na Parte Aérea

| Sintoma | Descrição |
|---------|-----------|
| **Murcha** | Planta murcha MESMO com solo úmido |
| **Folhas amarelando** | Começa pelas mais velhas (base) |
| **Folhas caindo** | Queda prematura, sem secar primeiro |
| **Crescimento parado** | Planta estagnada |
| **Caule amolecido** | Base do caule fica mole |
| **Cheiro ruim** | Odor de podridão do solo |

### O Teste Definitivo: Examinar as Raízes

```
Como verificar:
1. Retire a planta do vaso com cuidado
2. Observe as raízes
3. Compare com raízes saudáveis:

Raízes saudáveis:
✅ Brancas ou creme
✅ Firmes
✅ Cheiro de terra

Raízes podres:
❌ Marrons ou pretas
❌ Moles e se desfazem
❌ Cheiro de podre
❌ "Escorregam" se você puxar
```

## Podridão de Raiz vs Falta de Água

Este é o erro fatal! Os sintomas são parecidos:

| Característica | Podridão de Raiz | Falta de Água |
|----------------|------------------|---------------|
| Solo | Úmido ou encharcado | Seco |
| Murcha | Constante | Melhora após regar |
| Folhas | Amarelas E caindo | Murchas, depois secas |
| Caule | Pode estar mole na base | Firme |
| Cheiro | Podre | Normal |
| Raízes | Escuras, moles | Claras, secas |

**Regra de ouro:** Antes de regar planta murcha, VERIFIQUE O SOLO!

## Causas da Podridão de Raiz

### 1. Excesso de Água (Principal!)

```
Problemas comuns:
- Regar todo dia "por garantia"
- Não deixar o solo secar entre regas
- Vaso sem furo de drenagem
- Prato sempre cheio de água
- Regar por cima sem verificar o solo
```

### 2. Má Drenagem

```
Causas de má drenagem:
- Vaso sem furos
- Furos entupidos
- Solo muito argiloso/compacto
- Falta de material drenante (argila expandida)
- Vaso muito grande para a planta
```

### 3. Solo Inadequado

```
Solos problemáticos:
- Terra pura de jardim (compacta)
- Substratos velhos (sem estrutura)
- Solo sem matéria orgânica (encharca)
```

### 4. Vaso Grande Demais

Planta pequena em vaso grande = solo fica úmido por muito tempo = raízes afogam.

### 5. Fungos Oportunistas

Fungos como Pythium e Phytophthora vivem no solo e atacam raízes enfraquecidas por excesso de água.

## Como Salvar uma Planta com Podridão

### Passo 1: Diagnóstico

```
1. Retire a planta do vaso
2. Sacuda a terra para expor raízes
3. Avalie a extensão do dano:

Leve: Algumas raízes marrons, maioria saudável
Moderado: 50% das raízes comprometidas
Severo: Maioria das raízes podres
Terminal: Quase nenhuma raiz saudável + caule podre
```

### Passo 2: Poda de Raízes

```
Com tesoura esterilizada (álcool 70%):
1. Corte TODAS as raízes escuras/moles
2. Corte até encontrar tecido branco/firme
3. Não tenha pena - raiz podre não recupera
4. Descarte as partes podres no lixo
```

### Passo 3: Tratamento Antifúngico

```
Opções caseiras:
- Mergulhe raízes em solução de canela (1 col/sopa por litro)
- Ou polvilhe canela em pó nas raízes
- Ou use água oxigenada 3% por 5 minutos

Opções comerciais:
- Fungicida à base de cobre
- Trichoderma (fungo benéfico)
```

### Passo 4: Deixe Secar

```
Após tratamento:
1. Deixe a planta fora do vaso por algumas horas
2. Raízes devem secar levemente ao ar
3. Não deixe ao sol direto (estresse)
```

### Passo 5: Replante

```
1. Use vaso limpo (lave com água sanitária se reutilizar)
2. Vaso do tamanho certo (proporcional à planta/raízes)
3. FUROS DE DRENAGEM obrigatórios
4. Camada de argila expandida no fundo
5. Substrato novo e bem drenado
6. Plante e NÃO regue imediatamente
7. Espere 2-3 dias para primeira rega
```

### Passo 6: Cuidados Pós-Resgate

```
Próximas semanas:
- Mantenha em local iluminado mas sem sol direto
- Rega mínima - só quando solo secar
- Não adube até planta mostrar crescimento novo
- Paciência - recuperação leva semanas
```

## Taxas de Sucesso no Resgate

| Estágio | Chance de Recuperação |
|---------|----------------------|
| Leve (poucas raízes afetadas) | 90% |
| Moderado (50% raízes) | 60-70% |
| Severo (maioria das raízes) | 30-40% |
| Terminal (sem raízes + caule podre) | Menos de 10% |

**Quanto antes detectar, maiores as chances!**

## Prevenção: Os 7 Mandamentos

### 1. Vaso com Drenagem

```
Obrigatório:
- Furos no fundo do vaso
- Não tampe os furos
- Se usar cachepô decorativo, tire para regar
- Esvazie prato após 30 minutos
```

### 2. Substrato Drenante

```
Mistura ideal para vasos:
- 2 partes de substrato pronto
- 1 parte de perlita ou areia grossa
- Camada de argila expandida no fundo (3-5cm)
```

### 3. Teste Antes de Regar

```
Teste do dedo:
1. Enfie o dedo 2-3cm no solo
2. Está úmido? NÃO regue
3. Está seco? Pode regar

Teste do palito:
1. Espete palito de dente no solo
2. Retire após 10 min
3. Saiu úmido? Não regue
4. Saiu seco? Pode regar
```

### 4. Vaso do Tamanho Certo

```
Regra:
- Vaso só 2-3cm maior que o torrão
- Planta pequena = vaso pequeno
- Quando raízes saírem pelo furo, aumente
```

### 5. Não Deixe Água Parada

```
Elimine:
- Prato sempre cheio
- Cachepô com água acumulada
- Solo encharcado por dias
```

### 6. Boa Circulação de Ar

```
Ambiente arejado:
- Evita umidade excessiva
- Acelera secagem do solo
- Dificulta fungos
```

### 7. Rotina de Inspeção

```
Semanalmente:
- Verifique a umidade do solo
- Observe sinais de murcha
- Cheire o solo (podridão tem cheiro)
- Veja se há mosquitinhos (indicam excesso de umidade)
```

## FAQ - Perguntas Frequentes

### Como sei se estou regando demais?

Sinais: Solo sempre úmido, mosquitinhos fungus gnats, folhas amarelando da base, musgo verde na superfície do solo.

### Canela realmente funciona?

Sim! Canela tem propriedades antifúngicas comprovadas. Não é milagre, mas ajuda.

### Posso salvar uma planta sem raízes?

Algumas plantas podem ser propagadas por estaca se o caule ainda estiver saudável. Corte acima da podridão e tente enraizar em água ou substrato.

### Água oxigenada não queima as raízes?

Em concentração baixa (3%), não. Água oxigenada libera oxigênio, que ajuda raízes e mata patógenos anaeróbicos.

### Como evitar fungus gnats (mosquitinhos)?

Deixe a camada superior do solo secar. Eles reproduzem em solo úmido. Use areia grossa na superfície como barreira.

### Vaso de barro é melhor que plástico?

Para drenagem, sim. Barro é poroso e permite evaporação lateral. Plástico retém mais umidade.

## Sinais de Recuperação

Após o resgate, observe nas próximas semanas:

```
Sinais positivos:
✅ Murcha diminui
✅ Folhas param de cair
✅ Novas folhas surgem
✅ Crescimento retoma
✅ Planta firma ao toque

Sinais negativos:
❌ Murcha continua
❌ Mais folhas caem
❌ Caule amolece mais
❌ Cheiro de podre persiste
❌ Nenhuma melhora em 2 semanas
```

---

## Erros Comuns no Resgate

### Erro 1: Regar Após Replantar

A planta já está com raízes danificadas. Mais água piora. Espere 2-3 dias.

### Erro 2: Não Cortar o Suficiente

Raiz marrom não recupera. Corte tudo que estiver comprometido.

### Erro 3: Reutilizar o Mesmo Solo

Solo contaminado com fungos. Use substrato novo.

### Erro 4: Colocar no Sol Direto

Planta em recuperação está estressada. Sol indireto até melhorar.

### Erro 5: Adubar Imediatamente

Raízes danificadas não absorvem bem. Adube só quando houver crescimento novo.

## Resumo: Ação Imediata

```
Se você suspeita de podridão de raiz AGORA:

1. PARE de regar
2. Retire a planta do vaso
3. Examine as raízes
4. Se houver podridão:
   - Corte raízes podres
   - Trate com canela
   - Replante em substrato novo
   - Não regue por 2-3 dias
5. Monitore recuperação
```
